How to Edit This Page

To add a new game: Find the gamesData array in the JavaScript section and add a new object with properties: name, description, category, era, difficulty, image, rules, equipment, and cultural.

To change colors: Modify the CSS variables at the top of the :root section.

To edit text: Update the content directly in the HTML sections or in the gamesData array.

This is a self-contained HTML file - all code is in one place for easy editing!
